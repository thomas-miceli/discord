### Set config

```
$ npm install
```

Edit the [.env](https://github.com/thomas-miceli/discord/blob/master/.env) file :
* Add to the `SECRET` var the bot secret token.


### Add events

```shell
$ npm run new <command_name>
```


### Write code

Write code in the new method ```action()```


### Start the bot

```shell
$ npm run start
```
